📋 応募・入会について

入会するのに費用はかかりますか？

費用は一切かかりません。RVSSへの入会・活動はすべて無料です。応募料や月会費なども発生しません。

どうやって応募すればいいですか？

応募方法は主に以下の2つです。

  • 公募：SNS（Instagram等）やイベント等を通じて公式LINEからエントリー
  • リファラル（紹介）：既存メンバーからの紹介

流れ：合同説明会参加フォーム → 合同説明会参加 → 書類選考 → 面接（オンライン） → 合否連絡 → キックオフ・オンボーディング開始

いつでも入会できますか？募集時期はありますか？

通年で募集しています。特に4月の新歓シーズンは積極的に採用を行っていますが、時期を問わずエントリー可能です。

面接前に準備しておくことはありますか？

以下を事前にチェックしておくと、面接がより充実したものになります。

  • REAL VALUEのYouTubeチャンネルを視聴して、団体の雰囲気を掴む
  • RVSSのSNS（X / Instagram）をフォローして活動内容を確認する
  • 「RVSSで挑戦してみたいこと」を簡単に考えておく

必須ではありませんが、準備しておくと面接での会話がスムーズになります。

他のサークルやインターンと掛け持ちできますか？

掛け持ち自体は可能です。ただし、RVSSでは週15時間の稼働と定例MTGへの参加が求められます。他の活動との両立が現実的に可能かどうか、応募前にスケジュールを確認しておくことをおすすめします。

友達と一緒に入ることはできますか？

はい、リファラル（紹介）での応募も歓迎しています。ただし、選考は一人ひとり個別に行われます。友達同士でも、それぞれが面接を受けて合否が決まります。

🏠 RVSSってどんなところ？

そもそもRVSSって何をしている団体ですか？

REAL VALUE STUDENTS（RVSS）は、経営者コミュニティ「REAL VALUE」の学生支部です。

「日本をもう一度挑戦者に」というビジョンのもと、学生が実際のプロジェクトに参画しながら実践的なスキルを身につけるコミュニティです。

どんな部署・チームがありますか？自分はどこに入るんですか？

RVSSには主に以下のような部署があります。

  • 人事（採用・育成・評価・企画）
  • 広報（SNS運用・コンテンツ制作）
  • 総務（情報システム、財務・法務、OB会）
  • 各プロジェクトチーム（RVC、NoBorder、アカデミアなど）

入会直後にいきなり配属されるわけではありません。約1ヶ月の試用期間を経て、希望と適性をもとに部署が決まります。

関東・関西など支部があるようですが、自分はどこに所属しますか？

所属大学のエリアに基づいて支部が決まります（関東・関西・中部・九州など）。活動はオンラインが中心なので、エリアをまたいだ連携も日常的にあります。

メンバーは何人くらいいますか？どんな大学の人がいますか？

全国各地の大学から約40名以上の学生が参加しています。慶應・早稲田・立命館・京大・上智・明治・名古屋・神戸・九州大・長崎大など、多種多様な大学のメンバーが在籍しています。

学年もさまざまで、1年生から大学院生まで幅広く活動しています。

🚀 入会後、最初に何をすればいい？

入会してから正式に活動するまでの流れを教えてください

以下のステップで進みます。

  1. 面接 → 合格
  2. オンボーディング（約1ヶ月）
  3. 希望部署PMとの1on1 → 部署・チームアサイン

試用期間中にRVSSの文化を理解しながら、自分に合ったポジションを見つけていきましょう。

⏰ 稼働時間・MTGについて

週にどのくらい活動する必要がありますか？忙しいですか？

目安は週15時間です。

大学の授業やバイトと両立しているメンバーがほとんどです。MTGはオンラインが中心なので、場所を選ばず参加できます。稼働時間は「AI管理表」というNotionのデータベースで管理します。

定例MTGは週に何回ありますか？全部出ないとダメですか？

全体定例は土曜9:25〜11:00で、全メンバー必須です。その他、各部署の定例は所属部署やプロジェクトによって変わります。

部署や役職によっては複数の定例MTGに参加するメンバーも多いです。

どうしてもMTGに出られない場合はどうすればいいですか？

事前に連絡し、事情を伝えることで欠席が認められる場合があります。無断欠席は評価に大きく影響します。事前連絡を徹底しましょう。

💬 使用ツールについて

連絡・情報管理はどのツールを使っていますか？

LINE・Slackを使っています。情報管理のツールとしてNotion、Googleスプレッドシートを使っています。

Slack・Notionを使ったことがないのですが、大丈夫ですか？

大丈夫です。入会後にSlack・Notion・議事録研修があり、基本的な使い方を学べます。また、Notionの操作方法はRVSSのナレッジページにもまとめられています。

📊 評価・報酬について

どうやって評価されるんですか？

RVSSでは主に以下の仕組みで評価されます。

  • 360度評価：メンバー同士で相互評価し、行動習慣を数値化
  • AI管理表：タスクの完了状況・稼働時間・成果物を記録
  • 出席率：MTGへの参加状況も評価の一要素

「スキル」だけでなく、「姿勢」や「周囲への影響力」も重視されます。

報酬（給与）はもらえますか？いつからですか？

基礎研修をすべて完了した後、固定給が支給されます。入会直後の試用期間・研修期間は無償です。

基礎給に加え、360度評価スコア・出席率に応じて変動します。スクール受講完了による加算もあります。

頑張った分だけ報酬に反映される仕組みです。まずは研修をしっかり完了させましょう。

スキルアップのための研修やスクールはありますか？

はい、試用期間後も以下のようなインプット機会があります。

  • AIスクール
  • SNSスクール
  • 動画編集スクール
  • RVSS独自の育成カリキュラム

🤔 よくある不安・心配ごと

大学の授業やバイトと両立できますか？

多くのメンバーが大学生活と両立しています。MTGはオンライン中心で、場所を選ばず参加できます。ただし、スケジュール管理は自分の責任です。無理のない範囲で計画的に活動しましょう。

地方に住んでいても参加できますか？

はい、全国どこからでも参加できます。活動はオンラインが中心なので、場所は関係ありません。実際に北海道から九州、海外大学の学生もメンバーとして活躍しています。

特別なスキルがなくても大丈夫ですか？

大丈夫です。RVSSは「スキルがある人が入る場所」ではなく、「スキルを身につけていく場所」です。研修やスクール、実際のプロジェクト参加を通じて段階的にスキルを習得できます。

大切なのは、積極的に学ぶ姿勢と行動力です。
